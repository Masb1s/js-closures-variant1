Проєкт містить реалізацію практичних робіт 9, 9.1 та 9.2 (варіант 1) з теми замикань у JavaScript. Зроблено: базові лічильники (createCounter, createLimitedCounter, createStepCounter), іменований лічильник з історією (createNamedCounter), мемоізація (memoize, memoizeWith, memoizeAsync, memoizeExpiring), LRU‑кеш, статистика hits/misses, тести на Fibonacci, factorial та імітацію API. Усі приклади працюють у браузері через index.html. Приклади використання: const c = createCounter(5); c.increment(); c.get(); const lc = createLimitedCounter(0,3); lc.increment(); const sc = createStepCounter(2); sc.inc(); const memoFib = memoize(fib); memoFib(40); const memoFact = memoizeWith(factorial, n => "fact:"+n); memoFact(10); const memoApi = memoizeAsync(fetchUser); await memoApi(1); const ttlMemo = memoizeExpiring(x => x*x, 2000); ttlMemo(5)
